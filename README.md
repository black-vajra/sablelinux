# SableLinux

SableLinux is a custom Linux distribution built on LFS 12.4-systemd, targeting 
security research, penetration testing, OSINT, AI/LLM workflows, and virtualization.

> **Active development is on the [`development`](../../tree/development) branch.**  
> This branch (`main`) serves as a stable reference point.
